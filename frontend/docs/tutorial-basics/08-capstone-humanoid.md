

# Capstone Project — Autonomous Humanoid

Your final project combines everything:

 Voice command  
 Vision  
 Navigation  
 Object detection  
 Manipulation  
 ROS 2 control  

---

#  Goal

Build a humanoid that:

1. Hears a voice command  
2. Converts it into an action plan  
3. Navigates automatically  
4. Detects objects  
5. Picks them up  

---

# Architecture

Whisper → LLM → Planner → ROS Nav2 → Perception → Control


---

# Summary

This project demonstrates full Physical AI integration.
