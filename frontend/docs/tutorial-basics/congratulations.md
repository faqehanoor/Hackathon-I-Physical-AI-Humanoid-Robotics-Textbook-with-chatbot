
# Vision-Language-Action (VLA)

VLA connects:

- Vision (see)  
- Language (understand)  
- Action (respond physically)  

"Pick up the red cup" → Vision → Language → Action Plan


---

# Example with LLM

```text
Task: Clean the table
Plan:
1. Identify objects
2. Move to object
3. Pick it up
4. Place it in bin

Exercises

Write an action plan for: “Organize the books on the shelf.”

Summary

VLA connects intelligence with robotics.
